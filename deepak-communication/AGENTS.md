# Agent Instructions

This repository contains reusable communication profiles and communication-related Codex skills.

- Profiles define the speaker's voice.
- Channel overlays define differences between Teams, email, and generic chat.
- Skills define task workflows.
- Shared profile rules must not be copied into individual skills.
- No skill may send, post, reply, react, or modify browser content without explicit user instruction.
- Any claim about a person's contribution must be supported by visible conversation evidence.
- When evidence is weak, mark the attribution as uncertain.
- Never infer contribution importance from seniority, job title, or message count.
- Preserve privacy and inspect only the user-selected conversation.
